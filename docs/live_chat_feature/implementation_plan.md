# ライブチャットおよびスーパーチャット（投げ銭）機能の実装計画

## 概要
YouTube LiveやTikTok Liveのように、配信映像（プレースホルダー）の横にリアルタイムでコメントが流れる「ライブチャット機能」と、既存のトークン（TFT）を消費して目立つコメントを送信できる「スーパーチャット（投げ銭）機能」を実装します。ニコニコ動画のように画面上をコメントが横切るアニメーション（弾幕）も実装し、Web3ならではのトークンエコノミーを体験できるエンタメ要素を追加します。

## ユーザー確認事項 (User Review Required)
> [!IMPORTANT]
> **1. リアルタイム通信のバックエンドについて**
> 現在のアプリはスマートコントラクトとフロントエンドのみで構成されているようです。
> コメントを全ユーザー間でリアルタイム同期させるためには、FirebaseやSupabaseなどの外部データベース、もしくはWebSocketサーバーが必要です。
> 今回の実装では、**「まずはプレゼン・デモ用途としてUIとアニメーションを完全に作り込み、他人のコメントは自動生成（モック）で流し、自分のコメントやスーパーチャットが画面に反映されるフロントエンド完結の仕組み」**からスタートする形でよろしいでしょうか？（もし既にFirebase等をお使いであれば連携可能です）
> 
> **2. スーパーチャットの支払い先**
> スーパーチャット実行時にTFTトークンを消費（転送）します が、送信先は「配信者のアドレス（固定のダミーアドレス）」とするか、または「バーン（消失）させる」形のどちらがよいでしょうか？今回はデモ用の固定アドレスへの`transfer`を想定します。

#### [MODIFY] `src/pages/navbar/navbar.jsx`
- ナビゲーションメニューに「ライブ配信」へのリンクを追加します。

### ライブ配信ページ (Live Page)
#### [NEW] `src/pages/live/live.jsx`
- ページのメインレイアウト。左側に動画プレイヤー（YouTubeのiframe等）、右側にチャット欄を配置します。
- 画面上にコメントが流れる（弾幕）オーバーレイコンポーネントを組み込みます。

#### [NEW] `src/pages/live/live.css`
- ライブ画面専用のスタイリング、コメントが右から左へ流れるCSSアニメーション（`@keyframes`）、スーパーチャット用のリッチな装飾（グラデーション、ハイライト）を定義します。

### チャット・コメントコンポーネント
#### [NEW] `src/pages/live/components/chat_feed.jsx`
- 通常のコメントやスーパーチャットが縦にスクロールしていく履歴エリア。

#### [NEW] `src/pages/live/components/chat_input.jsx`
- コメント入力フォーム。
- 「通常送信」ボタンと、TFTトークンを消費する「スーパーチャット」ボタンを配置。

## 検証計画 (Verification Plan)
### 手動検証 (Manual Verification)
1. MetaMaskでログインし、ナビゲーションから「ライブ配信」ページへ遷移できることを確認する。
2. 動画プレイヤー上に、ダミーのコメントが右から左へアニメーションしながら流れることを確認する。
3. チャット入力欄から通常のテキストを送信し、チャット履歴と画面上の弾幕に即座に反映されることを確認する。
4. 「スーパーチャット」ボタンを押し、金額を指定して送信する。MetaMaskの署名ポップアップが開き、トランザクションが成功すると、特別な装飾（色付き、長めの表示時間）で画面に大きく表示されることを確認する。
5. MetaMaskまたはアプリ内の残高照会で、TFTトークンが正しく消費されていることを確認する。
